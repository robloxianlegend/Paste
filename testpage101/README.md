# Globalize Chat

A real-time chat application with reactions, mentions, and audio features.

## Features

- Real-time messaging
- User mentions with autocomplete
- Emoji reactions
- Message editing and deletion
- Image uploads
- Background music
- Responsive design

## Setup

1. Clone the repository
2. Open index.html in a web browser
3. Start chatting!

## Technologies Used

- React 18
- WebsimSocket
- RemixIcon
- GitHub Pages
